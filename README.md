1. **Weather Recommendation Program**
   - Prompts the user for the weather (sunny/rainy/cold) and provides clothing recommendations.

2. **Simple Calculator with Match Case**
   - Prompts the user for two numbers and an operation (+, -, *, /), calculates, and displays the result.
